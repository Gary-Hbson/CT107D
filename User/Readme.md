1.	问题：数码管在按下按键时闪烁
	因为：按键读取后一直重复调用DisNum()函数显示。缓冲区被快速檫除有重写导致
	解决：在显示之前先判断，使显示函数只执行一次
	
2.	没有按键按下时串口不能正常发送数据
	原因：未知
	解决：未解决
	
3.ds18B20给的驱动需要改时间，添加一个For循环，有时会读取255或127，过滤后显示

4. 使用按键松手检测和按下检测时，不能将触发标志位清空，不然程序之执行一次

5. DS1302向寄存器写一字节数据函数，输入形参的数据需要将十进制转换为十六进制
	读函数需要将返回的十六进制转换为10进制
	
6.注意在使用定时执行任务时，需要加标志位时任务只执行一次，否则会导致在1ms的时间内反复执行

7. 使用led时把led状态变量取反
